AnalyzeData Class

 

Namespace: LuxorTech.CreditSharp.Analyze

    

 

The AnalyzeData type serves at the root class used to generate aggregated credit variables from a normalized or merged data set.  An object is created by passing in one or more AnalysisBundle references, whee each bundle contains definitions for a group of related evaluations.  Execution is carried out using the Evaluate() method with the computed outcomes for each bundle being access through the Results property.

Example

 

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Analyze;

using LuxorTech.CreditSharp.Datamart.Common;

 

namespace HelpDemos

{

        public void ShowResults(CommonData common)

        {

          List<AnalysisBundle> bundles = new List<AnalysisBundle>();

          // Add new bundle using default option settings

          bundles.Add(new AnalysisBundle(AnalysisId.SummaryData01, Partitioning.IdMatch, new AnalyzeOptions()));

 

          AnalyzeData analyze = new AnalyzeData(bundles);

          analyze.Evaluate(common);

 

          // Code sample grabbing attributes via a numeric index

          foreach (AnalyzeResult result in analyze.Results)

          {

              for (int index = 0; index < result.Evaluations.Count; index++)

              {

                  Debug.WriteLine(string.Format("Attribute ['{0}'] has value of ['{1}']",

                                     result.Item(index).Name,

                                     result.Item(index).Value));

              }

          }

 

          // Code sample grabbing attributes using a literal name

          foreach (AnalyzeResult result in analyze.Results)

          {

              Debug.WriteLine(string.Format("Attribute ['TR0120'] has value of ['{0}']",

                                   result.Item("TR0120").Value));

              Debug.WriteLine(string.Format("Attribute ['TR0321'] has value of ['{0}']",

                                   result.Item("TR0321").Value));

          }

 

          // Code sample grabbing attributes using traditional list iterator

          foreach (AnalyzeResult result in analyze.Results)

          {

              foreach (AnalyzeAttribute attribute in result.Evaluations)

              {

                  Debug.WriteLine(string.Format("Attribute ['{0}'] has value of ['{1}']",

                                     attribute.Name, attribute.Value));

              }

          }

 

          return;

        }

}

Results

List

A collection of AnalyzeResult objects.

Contains the outcome(s) corresponding to each AnalysisBundle that was provided during object construction.  Each bundle may contribute more than one AnalyzeResult instance to this collection, depending what combination of Partitioning and IsolateJoint settings were assigned to bundle(s).  

Refer to the Remarks section of the AnalysisBundle topic page for more detail.
