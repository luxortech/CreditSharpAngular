TransUnionRequest.AddModel Method

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The AddModel method is used to configure and insert a request for a bureau scoring model into a credit bureau inquiry.  The method returns an instance of a BureauScore object for the newly created score model request.  With this reference, other properties of the model request (such as custom inputs, enabling consumer score disclosure features, etc) can be set.

 

Syntax

public BureauScore AddModel (String modelId)

 

Parameters:

 

modelId

Type: string

Contains the bureau specific id for the model being requested.

 

Exceptions

 

Exception class

Status

ParameterException

ParameterStatus.ElementAlreadyExists

ParameterStatus.ElementOutOfRange

Example

 

      public void SetEFxModel(string modelId, EquifaxRequest request)

      {

          // Equifax models need no other settings, unless custom inputs are required

          BureauScore newModel = request.AddModel(modelId);

      }

 

      // A hypothetical Equifax model which requires the consumer's employment income as an additional input

      public void SetCustomEfxModel(string modelId, EquifaxRequest request, string applicantIncome, ApplicantCardinality whichConsumer)

      {

          BureauScore newModel = request.AddModel(modelId);

           newModel.AddInput(whichConsumer, "INCOME", applicantIncome);

      }

 

      public void SetExpModel(string modelId, ExperianRequest request)

      {

          BureauScore newModel = request.AddModel(modelId);

           newModel.DisclosureType = ScoreDisclosure.Mortgage;

      }

 

      public void SetTucModel(string modelId, TransUnionRequest request)

      {

          BureauScore newModel = request.AddModel(modelId);

           newModel.SupplementalData = ScoreSupplement.RangeAndRank;

      }
