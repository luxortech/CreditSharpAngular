EvaluateTradeline Class

 

Namespace: LuxorTech.CreditSharp.Analyze

    

 

The EvaluateTradeline class is used to create a custom business rule which will examine each ComTradeline instance and report one of two outcomes:

 

•a true/false result to indicate whether or not a desired data condition (a predicate) is present

•a computed value based upon the data content using the provided formula (a solution)

 

Evaluations perform three types of logical actions:

 

•search for specific data conditions  ( when only a predicate is supplied )

•compute a specific value on each segment that exists  ( when only a solution is supplied )

•compute a specific value on each segment that matches a specific data condition  ( when both a predicate and solution are supplied )

 

Evaluations are added to an AnalysisBundle for eventual execution and once calculated will result in the creation of a corresponding AnalyzeAttribute instance.

 

Constructors

 

Name

Description

EvaluateTradeline(string, AnalysisOperation, ComTradelineIds, TradelinePredicate)

Creates a new evaluation which only defines a predicate to report how many instances of a specific data condition exist (a "where" clause).

EvaluateTradeline(string, AnalysisOperation, ComTradelineIds, TradelineSolution)

Creates a new evaluation which computes a value using the provided solution on every segment.

EvaluateTradeline(string, AnalysisOperation, ComTradelineIds, TradelinePredicate, TradelineSolution)

Creates a new evaluation which computes a value on each segment that matches a specific data condition.

The string parameter provides a unique name for the current evaluation. This name will be used to identify the computed value in all analysis outputs. As well, it can be used to locate the corresponding AnalyzeAttribute result using AnalyzeResult class's Item method.

Evaluations are constructed using delegates for the predicate and solution. Each can be defined using inline lambda expressions or compiled functions (or methods).  The examples below illustrate how to implement custom evaluations using each approach.  In general, a function delegate will execute faster than a lambda expression whereas lambda expressions may be quicker to develop.  As always, so some testing should be done to determine which approach is most appropriate for a given environment.

 

Lambda Implementation

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Analyze;

using LuxorTech.CreditSharp.Datamart.Common;

 

namespace HelpDemos

{

  public class PeformAnalysis

   {

      public AnalyzeData Execute(CommonData common)

       {

                 AnalyzeOptions options = new AnalyzeOptions();

                 AnalysisBundle bundle = new AnalysisBundle(AnalysisId.CustomEvaluation, Partitioning.IdMatch, options, "MyCustomAnalysis");

 

                //

                // Statements can be defined using separate declarations to contain each lambda expression  

                //

                TradelinePredicate isOpenTrl = (trade, options) => trade.IsOpen && !trade.IsCollection && !trade.EMPTY(ComTradelineIds.DateOpen);

                TradelineSolution mosSinceOpen = (trade, options) => trade.AGE(ComTradelineIds.DateOpen);

 

                // Then, as each evaluation is created, a reference to the distinct delegate expression is provided

                bundle.AddEvaluation(new EvaluateTradeline("MonthsSinceNewestOpen", AnalysisOperation.Minimum, ComTradelineIds.IdCount, isOpenTrl, mosSinceOpen));

                bundle.AddEvaluation(new EvaluateTradeline("MonthsSinceOldestOpen", AnalysisOperation.Maximum, ComTradelineIds.IdCount, isOpenTrl, mosSinceOpen));

 

                //

                // Alternatively, statements can be supplied as inline lambda expressions directly when creating an evaluation  

                //

                // (Note: these statements would yield the same computed values, they duplicated here purely for syntax demonstration purposes

                //

                bundle.AddEvaluation(new EvaluateTradeline("MonthsSinceNewestOpen2", AnalysisOperation.Minimum, ComTradelineIds.IdCount,

                                         (trade, options) => trade.IsOpen && !trade.IsCollection && !trade.EMPTY(ComTradelineIds.DateOpen), // the predicate

                                         (trade, options) => trade.AGE(ComTradelineIds.DateOpen)                                             // the solution

                                    );

                bundle.AddEvaluation(new EvaluateTradeline("MonthsSinceOldestOpen2", AnalysisOperation.Maximum, ComTradelineIds.IdCount,

                                         (trade, options) => trade.IsOpen && !trade.IsCollection && !trade.EMPTY(ComTradelineIds.DateOpen), // the predicate

                                         (trade, options) => trade.AGE(ComTradelineIds.DateOpen)                                             // the solution

                                    );

 

                //

                // Custom attributes can be constructed from predicates alone, solutions alone or both a predicate and a solution. The examples above use both

                // a predicate and solution.  The one below use one or the other.

                //

 

                // Predicate only

                TradelinePredicate averageBal = (trade, options) => !trade.IsCollection;

                bundle.AddEvaluation(new EvaluateTradeline("AverageBalance", AnalysisOperation.Average, ComTradelineIds.BalanceAmount, averageBal));

 

                // Solution only

                TradelineSolution averageAge = (trade, options) => trade.AGE(ComTradelineIds.DateReported) - trade.AGE(ComTradelineIds.DateOpen);

                bundle.AddEvaluation(new EvaluateTradeline("AverageAge", AnalysisOperation.Average, ComTradelineIds.IdCount, averageAge));

 

 

                //

                // Gather attributes into a list and compute the values

                //

                List<AnalysisBundle> bundles = new List<AnalysisBundle>();

          bundles.Add(bundle);

          AnalyzeData analyze = new AnalyzeData(bundles);

 

          analyze.Evaluate(common);

          return analyze;

       }

   }

}

 

 

Function Implementation

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Analyze;

using LuxorTech.CreditSharp.Datamart.Common;

 

namespace HelpDemos

{

  public class PeformAnalysis

   {

      public AnalyzeData Execute(CommonData common)

       {

          AnalyzeOptions options = new AnalyzeOptions();

                AnalysisBundle bundle = new AnalysisBundle(AnalysisId.CustomEvaluation, Partitioning.IdMatch, options, "MyCustomAnalysis");

 

                //

                // Each statement which defines a custom attribute is constructed using a compiled static function

                //

                bundle.AddEvaluation(new EvaluateTradeline("MonthsSinceNewestOpen", AnalysisOperation.Minimum, ComTradelineIds.IdCount, IsOpenTrade, MonthsSinceOpen));

                bundle.AddEvaluation(new EvaluateTradeline("MonthsSinceOldestOpen", AnalysisOperation.Maximum, ComTradelineIds.IdCount, IsOpenTrade, MonthsSinceOpen));

 

                //

                // Note that when only supplying one statement, either a predicate or solution, it is necessary to indicate which is being provided by using

                // a cast to the appropriate type:

                //

                bundle.AddEvaluation(new EvaluateTradeline("AvergageBal", AnalysisOperation.Average, ComTradelineIds.Balance, (TradelinePredicate)IsNotColl));

                bundle.AddEvaluation(new EvaluateTradeline("AvergageAge", AnalysisOperation.Average, ComTradelineIds.IdCount, (TradelineSolution)NbrOpenCol));

 

 

          List<AnalysisBundle> bundles = new List<AnalysisBundle>();

          bundles.Add(bundle);

          AnalyzeData analyze = new AnalyzeData(bundles);

 

          analyze.Evaluate(common);

          return analyze;

       }

 

            private static bool IsNotColl(ComTradeline trade, AnalyzeOptions options) { return (!trade.IsCollection); }

            private static bool IsOpenTrade(ComTradeline trade, AnalyzeOptions options) { return (trade.IsOpen && !trade.IsCollection && !trade.EMPTY(ComTradelineIds.DateOpen)); }

            private static double MonthsSinceOpen(ComTradeline trade, AnalyzeOptions options) { return (trade.AGE(ComTradelineIds.DateOpen)); }

            private static double AverageAge(ComTradeline trade, AnalyzeOptions options) { return (trade.AGE(ComTradelineIds.DateReported) - trade.AGE(ComTradelineIds.DateOpen)); }

   }

}
