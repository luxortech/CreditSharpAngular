LuxorTech.CreditSharp.Analyze Classes

 

    

 

Chapter Contents:

 

 

Name

Description

AnalysisBundle

Container class for selecting and configuring a set of attributes to be evaluated.

AnalyzeData

Root class used to perform selected calculations on a normalized credit bureau data image.

AnalyzeOptions

Container class for specifying various options for controlling Analyze features and functions.

AnalyzeResult

Container class for reviewing calculation results. A reference to this object can be passed to the FormatData type so that the values can be included in a predefined output.

AnalyzeAttribute

Class used to present the outcome of an individual calculation.

 

These Evaluate classes only need to be referenced if implementing custom Analysis bundles.

EvaluateIdentification

Class used to create and define a rule to be used for analyzing ComIdentification segment content.

EvaluateInquiry

Class used to create and define a rule to be used for analyzing ComInquiry segment content.

EvaluatePublicRecord

Class used to create and define a rule to be used for analyzing ComPublicRecord segment content.

EvaluateReference

Class used to create and define an analysis rule which references results of previously calculated rules.

EvaluateTradeline

Class used to create and define a rule to be used for analyzing ComTradeline segment content.
