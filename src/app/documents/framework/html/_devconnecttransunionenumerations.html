TransUnion Connect Enumerations

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The following tables list all the enumerated values available for use when requesting data from TransUnion.

 

Enumeration

Description

TucCpuResponse

Used to indicate if a CPU (automated) response is to be returned.

 

Member

Value

Description

None

0

No Cpu response desired.

Standard

1

TransUnion's default FFR layout will be returned.

 

 

TucCreditVisionAddOns

Used to indicate which, if any, optional CreditVision elements should be returned.

 

Member

Value

Description

None

0

No CreditVision add-ons desired.  (Default).

AddressHistory

1

Corresponds to the TUC service id 07220.

AccountHistory

2

Corresponds to the TUC service id 07225.

CreditSummary

4

Corresponds to the TUC service id 07226.

AccountSummary

8

Corresponds to the TUC service id 07227.

AllElements

15

Enabled the return of all four add-ons.

 

The AccountHistory enumeration is the prerequisite for populating Historical Activity displays using the TradelineDimensions option in Html layouts.

TucLanguageIndicator

Indicates what language should be used for remarks, headings, etc.

 

Member

Value

Description

English

0

Use English for all display elements. (Default).

Spanish

1

Use Spanish for all display elements.

French

2

Use French for all display elements (Canada only).

TucPointOfSale

Indicates if type of Point Of Sale transaction being conducted.

 

Member

Value

Description

None

0

Not a point of sale transaction. (Default)

General

1

Transaction is point of sale.

California

2

Transaction is point of sale and originating from California.

EmploymentReport

FraudDetect

IDSearch

Return the ID Search product.  (Service code: 07760)

IDReport

Return the ID Report product.  (Service code: 07770)

IdentityManager

Return the Identity Manager product.  (Service code: 06670)

SSNReport

SSNSearch

TucReferringBureau

Enumerates possible options for returning consumer referral contact information.

Member

Value

Description

None

0

Do not return any consumer referral contact information. (Default)

PhoneAndAddressOnly

1

Return the referring bureau's phone and address detail.

PhoneAddressAndURL

2

Return the referring bureau's website along with the phone and address.

TucRequestRegion

Indicates to which TransUnion region an inquiry should be directed.

Member

Value

Description

TestSystem

0

Submit the request to the test system (Default).

ProductionSystem

1

Submit the request to the production system.

TucTtyResponse

Used to indicate if a TTY (printable) response is to be returned.

Member name

Value

Description

None

0

No TTY response is desired.

ContinuousSheet

1

Include the default TTY report in the response.

 

(Only option supported when communicating to Canadian HostSystem)

CutSheetWithHeaders

2

Includes page breaks and numbering with column labels in header section

CutSheetNoHeaders

3

Includes page breaks and numbering

When requesting a TTY response from TU Canada, any non-zero enumeration will suffice as there is only one Canadian TTY version.

TucXmlResponse

Used to indicate if an XML response is to be returned.

Member

Value

Description

None

0

No XML response is desired.

Standard

1

The default XML response will be returned.
