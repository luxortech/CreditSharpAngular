ExpTradelineDimension Class

 

Namespace: LuxorTech.CreditSharp.Datamart.Experian

    

 

Defines an instance of an Experian Dimension class which can appear in a Tradeline segment. Together, a collection of ExpTradelineDimension instances can give a historical perspective on a number of different data elements.

 

The Experian Tradeline Dimensions property exposes a collection of ExpTradelneDimension instances.  Each instance corresponds to a specific date of the reported tradeline's history, as indicated in the EffectiveDate property.  The most recent month of available dimension detail will appear first, with each subsequent entry providing progressively older views of the account history.
