Valid Response Combinations

 

    

 

Equifax has a number of different response layouts in both human and machine readable forms.  With some products, the Equifax system has the ability to return multiple layouts with a single request, so that users can retrieve a machine readable copy for data analysis and human readable copy for visual review.

 

The following table lists which response layouts can be combined into a single request.  These combinations are requested by simply enabling the desired response enumerations.  For example, to request a Full File Fixed CPU and an Acrofile Plus TTY, use:

 

          EquifaxRequest request = new EquifaxRequest(options);

           request.CpuResponse = EfxCpuResponse.FullFileFixed;

           request.TtyResponse = EfxTtyResponse.PlusPrintImage;

 

 

Please note: The only EfxTtyResponse enumeration that can be used in conjunction with the new cloud-based EfxJsonResponse is EfxTtyResponse.PdfImage.  All other printable report layouts will no longer be available from the new cloud-based Equifax API.
