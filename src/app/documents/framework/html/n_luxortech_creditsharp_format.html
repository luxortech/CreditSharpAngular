LuxorTech.CreditSharp.Format Namespace

 

    

 

The Format namespace is where the various output options are implemented for extracting imported credit data from the CreditSharp framework.  The Format namespace presents four basic output types:

 

•Html

an interactive version of the Text layout with a more modern visual style

•Json

a structured output created in JavaScript Object Notation standard and intended for use with machine based processing

•Text

a static version of the Text layout which can be paginated for more attractive results when sent to a printer

•Xml

an XML structured output intended for suitable for machine based processing

 

Examples of each of these output formats can be reviewed in the Credit Sharp Outputs chapter.

 

An example of a typical programming pattern

 

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Datamart.Common;

using LuxorTech.CreditSharp.Format;

 

namespace HelpDemos

{

  public class DemoFormat

  {

      public string CreateHtml(CommonData common)

      {

          FormatOptions options = new FormatOptions(FormatLayout.Html);

          options.Html.Partitions = Partitioning.IdMatch;

          options.Html.ReportStyle = HtmlStyle.Common01;

          options.Html.TradelineStyle = HtmlTradelineStyle.Tradeline02;

 

          FormatData format = new FormatData(options);

          return format.Generate(common);

      }

   }

}

 

 

Format Classes

 

Name

Description

FormatData

Root class used to create formatted output from imported credit bureau data.

FormatGroup

A container object used to specific how separate groups within a given output section are decorated.

FormatOptions

Container class for indicating the options and settings that will control the creation of formatted output.

FormatOptionsHtml

Provides access to settings for creation of Html based formats.

FormatOptionsJson

Provides access to settings for creation of Json based formats.

FormatOptionsText

Provides access to settings for creation of Text based formats.

FormatOptionsXml

Provides access to settings for creation of Xml based formats.

FormatSection

A container object which controls which output sections are displayed and in what order.

 

Format Enumerations

 

Name

Description

FormatLayout

Enumerates the possible layouts for generated formatted credit data.

HtmlScoreRankDisplay

Indicates how a score's percentile rank will be shown in Html layouts.

HtmlStyle

Enumerates the possible Styles when generating Html based outputs.

HtmlTradeStatusSource

Indicates how a tradeline status display is chosen in Html layouts.

HtmlTradeStatusDisplay

Indicates if a separate status element should be displayed in the tradeline section of an Html layout.

HtmlTradelineStyle

Enumerates the possible Tradeline content layouts when the section is in its collapsed state.

JsonMergeHistory

Specifies if any and what type of merge history should be included in JSON output.

JsonStyle

Enumerates the possible Styles when generating JSON based outputs.

TextStyle

Enumerates the possible Styles when generating Text based outputs.

XmlMergeHistory

Specifies if any and what type of merge history should be included in Xml output.

XmlStyle

Enumerates the possible Styles when generating Xml based outputs.
