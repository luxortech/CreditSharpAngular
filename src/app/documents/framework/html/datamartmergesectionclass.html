MergeSection Class

 

Namespace: LuxorTech.CreditSharp.Datamart.Merge

    

 

The MergeSection class can be used to review of modify settings for specific sections of a CreditSharp CommonData input. The various settings can selectively modify merge behavior from the system default value(s) on each data segment.

 

Not all settings can be used with all segments.  As such, if an invalid combination is attempted an ElementOutOfRange exception will be thrown.  The Exceptions portion of this topic below details the possible valid settings that can be applied.
