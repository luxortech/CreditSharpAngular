ApplicantEmployer Type

 

Namespace: LuxorTech.CreditSharp.Core

    

 

The ApplicantEmployer members belong to a static class accessed via an instance of the Applicant class and are used to indicate an applicant's employment history.

 

Properties

 

Name

Content

Description

DateHired

Text (10)

The date the applicant began working for the employer.

 

Format:

Entered in month, day, year order. At minimum, the input must include the month and year.  It can contain standard date delimiters.  For example, March 16th 1997 can be input as:

03-1997, 031997, 03/1997, 03/16/1997, 03-16-1997, 03161997.

DateLeft

Text (10)

The applicant's last date of employment with this employer.

 

Format:

Entered in month, day, year order. At minimum, the input must include the month and year.  It can contain standard date delimiters.  For example, March 16th 1997 can be input as:

03-1997, 031997, 03/1997, 03/16/1997, 03-16-1997, 03161997.

The name of applicant's employer.
