MTG( Mortgage) Attributes

 

    

 

The Mortgage attribute category includes values calculated with references secured by real state.  This would include both first and second mortgages, some home improvement loans, etc.  Home Equity values are not included here, use HEQ for that information.  To obtain values which include both Mortgage and Home Equity, use REA.
