LuxorTech.CreditSharp.Datamart.Common Namespace

 

    

 

The Common namespace presents the classes and enumerations that together present a normalized view of any credit bureau response.  Using the Normalize method, a bureau data object is transformed into a collection of commonly defined credit data references. This allows for the same business rules and logic to be performed regardless of which credit bureau provided the underlying data.

 

The resulting normalized object can then be passed to the Merge, Analyze and Format namespaces for additional processing.

 

A typical programming pattern

 

using LuxorTech.CreditSharp.Datamart;

using LuxorTech.CreditSharp.Datamart.Common;

 

namespace HelpDemos

{

  public class DemoCommon

   {

      public CommonData NormalizeBureau(BureauData creditDataObject)

      {

          CommonOptions options = new CommonOptions();

          options.DateMask = "YYYYMM01";

          options.EstimateInstallmentPayments = true;

          options.EstimateRevolvingPayments = true;

 

          CommonData commonData = new CommonData(options);

          commonData.Normalize(creditDataObject);

          return commonData;

      }

   }

}
