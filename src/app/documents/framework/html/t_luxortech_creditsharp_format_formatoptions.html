FormatOptions Class

 

Namespace: LuxorTech.CreditSharp.Format

    

 

The FormatOptions class is a container for indicating the options and settings that will control the creation all formatted output.  An instance of the class is created and passed into the FormatData class constructor

Inherit

Bool

Gets or sets the value indicating whether certain option properties should be inherited from the data class being formatted.

 

When true, the values for the MaskSsn option is copied from the CommonData or MergeData object being formatted.

UseAlphaCurrentMop

Bool

Gets or sets the value indicating if current months within tradeline payment histories should be shown using a 'C' as opposed to the standard '0' or '1'.

 

[Defaults to false]

UseSatisfiedMop

Bool

Gets or sets the value indicating if a tradeline's current mop value should be designated as 'S' to signify 'Satisfied' for severely derogatory accounts has been reported as paid (having a zero balance).  This applies to collection, charge off, repossession or foreclosure accounts.

 

In Html outputs, the value will appear as 'STS' (instead of 'COL', for example).  In Text outputs, the value will appear as 'SATIS'.  Json and Xml outputs would contain an 'S' code in the CurrentMop element.

 

[Defaults to false]
