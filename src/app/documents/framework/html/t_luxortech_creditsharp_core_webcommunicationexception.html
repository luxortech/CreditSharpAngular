WebCommunicationException Class

 

    

 

The WebCommunicationException class is derived from the CreditSharpException base class and is used to report an error state that has been returned by some part of the operating system's communication sub-system(s).  This can perhaps be a problem within the WinHttp communication stack, the SSL encryption services, a proxy server error, and so on.

 

In addition to the Status value provided by the System.Net.WebExceptionStatus, the InnerException and Message properties can often provide further details about the point of failure.
