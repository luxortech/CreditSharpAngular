Merge Data Classes

 

Namespace: LuxorTech.CreditSharp.Datamart.Merge

    

 

The table below lists all the data classes which together make up a merged view of a credit bureau response.

 

 

Name

Description

MrgAddress

Defines an instance of a normalized consumer address segment.

 

Details of a consumer's address history will be found here.

MrgAlert

Defines an instance of a common bureau alert segment.

 

Provides a central location for detailed presentation of various types of alerts, such as fraud warnings, OFAC searches, etc.

MrgAlias

Defines an instance of a merged consumer alias segment.

MrgBanking

Defines an instance of a merged banking segment.

MrgBureauAttribute

Defines an instance of a merged credit bureau attribute segment.

MrgBureauError

Defines an instance of a merged credit bureau error segment.

MrgConsumerReferral

Defines an instance of a merged consumer referral segment.

MrgConsumerStatement

Defines an instance of a merged consumer statement segment.

MrgDirectory

Defines an instance of a merged subscriber directory segment.

MrgEmployment

Defines an instance of a merged consumer employment segment.

MrgFraudScan

Defines an instance of a merged fraud scan segment.

MrgHeader

Defines an instance of a merged header segment.

MrgIdentification

Defines an instance of a merged consumer identification segment.

MrgInquiry

Defines an instance of a merged credit bureau inquiry segment.

MrgMessage

Defines an instance of a merged message segment.

MrgNotification

Defines an instance of a merged notification segment.

 

Provides a central location for flags pertaining to certain noteworthy statuses, including Active Military Duty, Military Borrower Search, Address Variations, etc.

MrgOfacSearch

Defines an instance of a merged OFAC search segment.

MrgPublicRecord

Defines an instance of a merged public record segment.

MrgRegisteredItem

Defines an instance of a merged registered item segment.

MrgScore

Defines an instance of a merged bureau score segment.

MrgTradeline

Defines an instance of a merged tradeline segment.
