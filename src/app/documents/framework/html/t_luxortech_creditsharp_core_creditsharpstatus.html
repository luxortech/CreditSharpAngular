CreditSharpStatus Class

 

    

 

The CreditSharpStatus class can be used to transform EncodedStatus property values found in the various CreditSharp exceptions.  The class breaks out the four basic error elements from a packed status value: Domain, Region, Context and the Error Code (which corresponds to one of the namespace specific Exception Enumerations).

 

A decoded values can be obtained either by passing the status code into the constructor or by setting the EncodedStatus property after the fact.  It also possible to 'compute' a new EncodedStatus by setting each of the four error elements separately, although that is something that would not typically be required by eternal CreditSharp users.

 

The resulting decoded values can be obtained individually or as a formatted error string.

Parameters:

 

statusCode

Type: int

When provided, contains a status to be decoded.

 

Example 1

 

// Example using constructor to directly decode value

CreditSharpStatus status = new CreditSharpStatus(67108867);

Debug.WriteLine(status.ErrorText);

 

// Outputs

 

Domain  : CreditSharp

Region  : BureauCreation

Context : General

ErrorNbr: 3 (CreditDataUnidentified )

Example 2

 

// Example using property to decode value

string breakout;

CreditSharpStatus status = new CreditSharpStatus();

 

status.EncodedStatus = 67108867;

breakout = string.Format("Domain: {0} Region: {1} Context: {2} Error: {3}",

                        status.Domain, status.Region, status.Context, status.ErrorEnum);

Debug.WriteLine(breakout);

 

status.EncodedStatus = 574619699;

breakout = string.Format("Domain: {0} Region: {1} Context: {2} Error: {3}",

                       (int)status.Domain, (int)status.Region, (int)status.Context, status.ErrorEnum);

Debug.WriteLine(breakout);

 

// Outputs

 

Domain: CreditSharp Region: BureauCreation Context: General Error: CreditDataUnidentified

Domain: 1 Region: 1 Context: 4 Error: not available
