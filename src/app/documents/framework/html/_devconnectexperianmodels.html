Experian Score Models

 

    

 

 

The table below lists all Experian models currently recognized by the CreditSharp framework.

 

Users can also request any custom Experian models that may have been developed specifically for them.  To order a such a model, the prefix 'CM-' can be used to identify the desired item. For example, CM-XY would be used to request a hypothetical custom Experian model with an id of 'XY'.

 

CreditSharp uses Experian's NetConnect interface to order bureau data and this interface does not include support for every possible model.  The NetConnect Enabled column in the following table indicates if a given model can be requested with the inquiry.
