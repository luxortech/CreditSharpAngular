_format sections_parsed

  

Section Name

Description

Header

Displays the date the credit report was obtained, any reference number value and which bureaus are represented.

BureauError

Displays any error information reported by a credit bureau.

Identification

Displays the consumer name, age, and social security numbers.

Address

Displays the consumer's current and former residential address history.

Alert

Displays outcome of any fraud search services, active duty notices, etc.

Alias

Displays any alternate names consumer may be known by.

Summary

Displays a table with summary level elements.

ConsumerStatement

Displays any consumer statement information included in the bureau data.

Score

Displays the results of all bureau score model requests.

Employment

Displays the consumer's employment history when available.

FraudScan

Displays the outcome of any services which scan for occurrences of fraud or potential fraud markers.

PublicRecord

Displays any legal actions record against the consumer.

Tradeline

Displays details of all credit account and payment history.

Message

Displays general items that don't fit within other report categories.

Notification

Displays a overview of notices or warnings gathered from detailed reports within the response data.

Inquiry

Displays a record of all what other entities or business have checked the consumer's credit. history.

OfacSearch

Displays the outcome of an Office of Foreign Assets Control search.

ConsumerReferral

Displays the information for a consumer to use when contacting the bureau furnishing the data.

Directory

Displays the full name, address and phone number (as available) for all entities that reported information included in the output.
