TucInquiryOptions Class

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The properties presented within the TucInquiryOptions class are specific to a TransUnion credit bureau request.  They are used to request various forms of supplemental information be added to a credit report response.

Name

Description

TucInquiryOptions(string licenseKey)

Constructor for an instance of the TucInquiryOptions class.

public TucInquiryOptions(string licenseKey)

 

Parameters:

licenseKey

Type: string

The 36 character key which grants access to all CreditSharp features.
