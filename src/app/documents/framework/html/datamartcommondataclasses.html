Common Data Classes

 

Namespace: LuxorTech.CreditSharp.Datamart.Common

    

 

The table below lists all the data classes which together make up a normalized view of a credit bureau response.

 

 

Name

Description

ComAddress

Defines an instance of a normalized consumer address segment.

 

Details of a consumer's address history will be found here.

ComAlert

Defines an instance of a common bureau alert segment.

 

Provides a central location for detailed presentation of various types of alerts, such as fraud warnings, OFAC searches, etc.

ComAlias

Defines an instance of a normalized consumer alias segment.

ComBanking

Defines an instance of a normalized banking information segment.

ComBureauAttribute

Defines an instance of a normalized bureau attribute or characteristic found in custom scoring or decisioning solutions.

ComBureauError

Defines an instance of a normalized credit bureau error segment.

ComConsumerPhone

Container class for exposing various phone numbers associated with a given consumer.  

 

Instances only appear within the ComIdentification PhoneNumbers property.

ComConsumerReferral

Defines an instance of a normalized consumer referral segment.

ComConsumerStatement

Defines an instance of a normalized consumer statement segment.

ComDelinquency

Container class for exposing delinquency date detail on a given tradeline.  

 

Instances only appear within the ComTradeline Delinquencies property.

ComDimension

Container class for presenting a month of historical account status for a given tradeline.

 

Instances only appear within the ComTradeline Dimensions property.

ComDirectory

Defines an instance of a normalized subscriber directory segment.

ComEmployment

Defines an instance of a normalized consumer employment segment.

ComFraudScan

Defines an instance of a normalized fraud scan segment.

ComHeader

Defines an instance of a normalized header segment.

ComIdentification

Defines an instance of a normalized consumer identification segment.

ComInquiry

Defines an instance of a normalized credit bureau inquiry segment.

ComMessage

Defines an instance of a normalized message segment.

ComNotification

Defines an instance of a normalized notification segment.

 

Provides a central location for flags pertaining to certain noteworthy statuses, including Active Military Duty, Military Borrower Search, Address Variations, etc.

ComOfacSearch

Defines an instance of a normalized OFAC search segment.

ComPublicRecord

Defines an instance of a normalized public record segment.

ComRegsiteredItem

Defines an instance of a normalized registered item segment.

ComScore

Defines an instance of a normalized bureau score segment.

ComTradeline

Defines an instance of a normalized tradeline segment.
