Name

Description

Classes

BureauConnection

Class containing settings and methods related to management of a communication session with a given bureau.

BureauRequest

Base class from which each bureau specific request class is derived.

BureauScore

Class contains an instance of a bureau scoring model being requested.

BureauService

Class contains an instance of a custom or optional bureau feature to be requested.

BureauSubscriber

Provides access to the output from a subscriber search request.

HttpConfig

Container class for providing values related to credentials and web addresses associated with internet  credit bureau communications.

ProxyServer

Class used to specify settings when a proxy server is being used for bureau communications.

ConnectionType

Enumerates possible types of bureau communication protocols.

ProxyType

Enumerates the possible modes of proxy communication.

ScoreDisclosure

Used to specify if a consumer score disclosure form should be embedded in the Experian bureau response.

ScoreSupplement

Indicates if supplemental consumer score disclosure elements are to be included in the bureau response.
