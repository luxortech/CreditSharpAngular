VAR (Variable payment) Attributes

 

    

 

The Variable attribute category includes values calculated with references to all tradeline account types with payments determined by the outstanding balance, which can change on a monthly basis.  As such, these values include information from the Revolving, Credit Line and/or Open/30 Day account types.
