ExpTtyFormat

Used to indicate if a TTY (printable) response is to be returned

Member

Value

Description

Format01

0

Return Experian's default format. (Default)

Format02

1

Return Experian's alternate format.

The difference between each Trade Format option is primarily centered around the display of an account's payment history. With the Format01 option (Experian's default selection) a full 25 month payment pattern is shown. With the Format02 option only the first 13 months of the pattern are presented, with the extra space used to show the 30/60/90 delinquency counters. The Format02 option also adds an additional display column to the two most recent delinquency dates.

ExpTtyPaging

Indicates which paging scheme should be used in the TTY response.

Member

Value

Description

PagingOn

0

Use the default paging scheme. (Default)

PagingOff

1

Turn the default paging scheme off.

Experian has two basic options for controlling page layouts. When Paging option is enabled (Experian's default selection) the Trade section header is repeated on each page, as necessary, with page breaks and spacing inserted to conform to an actual printed page. When Paging is disabled the Tradeline header is only displayed one time at the top of the Trade section, even if tradelines are continued on subsequent pages and no page breaks or form spacing are included.
