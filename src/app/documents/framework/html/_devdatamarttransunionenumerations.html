The TucServiceType table indicates which basic TransUnion service provided the detail in a given segment.

 

TucServiceType


Member name

Value

Description

NotSpecified

0

No service type was specified.

Product

1

The segment(s) are part of the basic product fulfillment response.

Addon

2

The segment(s) were generated by the inclusion of an optional add-on service.

Score

3

The segment(s) provide additional detail to a returned score model.
