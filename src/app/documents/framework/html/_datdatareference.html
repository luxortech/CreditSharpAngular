Common codes and definitions

In addition to providing one set of universally applied naming conventions for accessing credit bureau information, regardless of the providing bureau, the Common and Merge namespaces also employ a single set of codes and definitions for various elements of the credit report.

 

Rather than having to create logic to recognize the possible codes and definitions that can be provided by all three credit bureaus, a process can be developed that references a single set of normalized values, which remain constant, regardless of the data source.
