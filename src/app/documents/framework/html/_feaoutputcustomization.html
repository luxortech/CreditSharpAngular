Customizing outputs

 

    

 

The content of each CreditSharp output layout can be manipulated to serve many different needs or meet customer preferences.

 

Each output layout has a number of predefined option settings (accessible through the FormatOptions class) which can alter the appearance of elements with the various report sections.  Descriptive text can be added to Json and Xml outputs, the visual composition of tradeline items can be customized in Html and Text outputs and entire sections can be suppressed or emitted in a different sequence.

 

In addition to the predefined settings, completely custom definitions can be used to alter the sort criteria of elements within given report sections as well as creating groups within sections when generating Html or Text layouts.

 

For complete discussions about these various options and customization opportunities, please refer to the detailed topic pages:

Minimize/Resequence Outputs

 

If so desired, an output could be modified to display only the tradelines that exist in the credit data response.  Or just the Tradelines and Inquiries.  Or any other combination.  

 

One of the more common choices it to suppress the Directory section of any output as that detail is not often used in today's credit approval environments and largely is wasted space (both visually and in terms of file size), with the following line of code:

 

          FormatOptions options = new FormatOptions(FormatLayout.Text);

           options.Text.Section("Directory").Visible = false;

          FormatData format = new FormatData(options);

 

The following effectively turns off display of all sections except Tradeline and Inquiry:

 

          FormatOptions options = new FormatOptions(FormatLayout.Text);

           options.Text.FirstSection("Tradeline");

           options.Text.NextSection("Inquiry");

          FormatData format = new FormatData(options);

 

It should be noted that the Header section is always displayed no matter what and will always be the first section.  The software does allow you to set it's visible property to false, but that setting will be ignored.
