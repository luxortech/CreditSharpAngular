Exclude Method

 

  

 

The Exclude method can be invoked on Identification, Inquiry, PublicRecord and Tradeline objects.  The method toggles the exclude state on or off, returning a Boolean value reflecting what state has been set.  The effect of this method can also been seen in the IsExcluded property of each object which returns the ExcludeState enumerated value.

 

Items explicitly excluded using this method will not be included in any AnalyzeData calculations nor will they be shown in any of the various Format outputs.  For this reason, care should be taken when setting up exclusion logic to ensure that the desired outcome is achieved and that items are not inadvertently hidden from view.  The exclude state can be turned of all manually excluded items at once with a call to the RestoreExcludes method.

 

Usage

 

One possible scenario might involve a business case where all known medical debts are eliminated from any evaluation of a consumer's creditworthiness.  The example shown  here demonstrates how this could be accomplished, with each debt being flagged by the Exclude method if they are medical obligations.  This means that any AnalyzeData or FormatData operations performed against that CommonData object would skip past each observation marked as 'excluded'.

 

Note: This state does not carry on to the MergeData object.  If items are excluded in the CommonData object, they would still be picked up during a merge operation.  If items within a MergeData object are to be skipped over, they would need to be excluded directly through that merge instance itself.

 

Syntax

public bool Exclude ()

 

Parameters: none

 

Example

 

using LuxorTech.CreditSharp.Datamart.Common;

 

namespace HelpDemos

{

  public class HideMedicalDebts

   {

      public void HideItems(CommonData commonData)

       {

          foreach (ComPublicRecord segment in commonData.PublicRecordSegments)

           {

              if (segment.IsMedical)

                   segment.Exclude();

           }

 

          foreach (ComTradeline segment in commonData.TradelineSegments)

           {

              if (segment.IsMedical)

                   segment.Exclude();

           }

       }

   }

}
