ApplicantIdentity Class

 

Namespace: LuxorTech.CreditSharp.Core

    

 

The ApplicantIdentity members belong to a static class accessed via an instance of the Applicant class and are used to describe an applicant's personal details.

 

Properties

 

Name

Content

Description

Age

Num (3)

Applicant's age.

DateOfBirth

Text (10)

The applicant's date of birth.

 

Format:

Entered in month, day, year order. At minimum, the input must include the month and year.  It can contain standard date delimiters.  For example, March 16th 1997 can be input as:

03-1997, 031997, 03/1997, 03/16/1997, 03-16-1997, 03161997.
