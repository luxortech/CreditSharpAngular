Evaluate Method

 

Namespace: LuxorTech.CreditSharp.Analyze

    

 

The Evaluate method is used to instruct the AnalyzeData object to perform the calculations on each evaluation item contained within the AnalysisBundle list passed the AnalyzeData object's constructor.  Once the operation is complete, the AnalyzeResult member of the AnalyzeData object will contain the outcomes of each computation.

 

Evaluations can be performed on both a normalized CommonData object or a MergeData object which contains a deduped view of the normalized credit data.

 

Method Overloads

 

Name

Description

Evaluate(CommonData commonInput)

Applies the evaluations found within the list of AnalysisBundle references against the normalized input.

Evaluate(MergeData mergeInput)

Applies the evaluations found within the list of AnalysisBundle references against the normalized and deduped input.

 

Syntax

public void Evaluate (CommonData commonInput)

public void Evaluate (MergeData mergeInput)

 

Parameters:

 

commonInput

Type: CommonData object reference

Contains the normalized credit data to be evaluated.

mergeInput

Type: MergeData object reference

Contains the normalized and deduped credit data to be evaluated.

 

Example

 

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Analyze;

using LuxorTech.CreditSharp.Datamart.Merge;

 

namespace HelpDemos

{

        public void AnalyzeMerged(MergeData mergedData)

        {

          List<AnalysisBundle> bundles = new List<AnalysisBundle>();

          // Add new bundle using default option settings

          bundles.Add(new AnalysisBundle(AnalysisId.SummaryData01, Partitioning.IdMatch, new AnalyzeOptions()));

 

          AnalyzeData analyze = new AnalyzeData(bundles);

          analyze.Evaluate(mergedData);

 

          return;

        }

}
