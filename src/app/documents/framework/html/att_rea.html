REA (Real Estate) Attributes

 

    

 

The Real Estate attribute category includes values calculated for all references secured by real state.  This would include both first and second mortgages, some home improvement loans, and any Home Equity accounts.  To see Mortgage and/or Home Equity values separately, use HEQ and/or MTG.
