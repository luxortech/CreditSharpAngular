Equifax Connect Enumerations

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The following tables list all the enumerated values available for use when requesting data from Equifax.

 

Enumeration

Description

EfxCanadaContent

Used to indicate if data unique to Canada should be filtered out or returned and if shared elements are defined using US or Canadian specifications.

 

Member

Value

Description

PassThru

0

Return US equivalent elements as well as unique Canadian elements.

Native

1

All elements conform to Canadian specifications.  

 

(Note: option is not currently supported)

Filtered

2

Only US equivalent elements are returned.

 

 

EfxCpuResponse

Used to indicate if a CPU (automated) response is to be returned.

 

Member

Value

Description

None

0

No Cpu response desired.

FullFileFixed

1

Equifax's Full File Fixed layout will be returned.

AcrofilePlus

2

Equifax's ACROFile Plus layout will be returned.

HumanMachineReadable

3

Equifax's Full File Fixed - Human Machine Readable layout will be returned.

 

 

EfxCreditView

Used to indicate which credit report variant is desired.  Applies only to API requests and is not used for Legacy transactions.

 

Member

Value

Description

Standard

0

Return the standard API credit report.

Limited

1

Return an abbreviated version of the standard API credit report.  

 

(Note: option is not currently supported)

OneView

2

Return the OneView credit report (which includes the Standard report along with optional additional data sources/outputs).

 

 

EfxJsonResponse

Used to indicate if what version of JSON response is to be returned.

 

Member

Value

Description

None

0

No JSON response desired.

Standard

1

Version 1.0 of the cloud-based API layout.

 

 

EfxMonthCount

Enumerates maximum age of inquiry observations to be returned.

 

Member

Value

Description

AllValues

0

Indicates that Equifax default should be used (currently 24 months).

ThreeMonths

1

Indicates inquiries are only to be counted from the past three months.

SixMonths

2

Indicates inquiries are only to be counted from the past six months.

NineMonths

3

Indicates inquiries are only to be counted from the past nine months.

TwelveMonths

4

Indicates inquiries are only to be counted from the past twelve months.

EfxProductCode

Enumerates possible bureau products which can be requested.

 

Member

Value

Description

Unspecified

0

No product code specified.

CreditReport

1

The primary response option suitable for making lending decisions.

DtecReport

2

The DTEC product is geared toward collection efforts, returning consumer identification detail.

FindersReport

3

The Finders report is a collection agency product, returning an abbreviated image of a consumer credit profile.

MilitaryBorrowerCheck

4

Performs a lookup against the Dept Of Defense database to determine if applicant is a covered military borrower.

RecoveryReport

5

The Recovery report is used for collections purposes  both skip tracing and assessing ability to pay.

PersonaReport

6

The Persona report is a tool for employers to screen job applicants.

 

EfxScoreSupplement

Indicates if supplemental consumer score disclosure elements are to be included in the bureau response.

 

Member

Value

Description

None

0

Do not return any Risk Based Pricing information.

BeaconScores

1

Include Risk Based Pricing detail on any Beacon (Fico) score model.

AllScores

2

Include Risk Based Pricing detail on all applicable models.

 

EfxTtyResponse

Used to indicate if a TTY (printable) response is to be returned

Member

Value

Description

None

0

No Tty response is desired.

StandardPrintImage

1

Indicates a request for the Standard Print Image (Equifax Output Format Code: 06)

AdvancedPrintImage

2

Indicates a request for the Advanced Print Image (Equifax Output Format Code: 16)

PlusPrintImage

3

Indicates a request for the Acrofile Plus Print Image (Equifax Output Format Code: 01)

AcrofilePlus

4

When paired with EfxCpuResponse.FullFileFixed, indicates an Acrofile Plus Image should be returned as well (Equifax Output Format Code: 32)

PdfImage

5

Indicates a request for a PDF image from the new cloud-based API.


EfxXmlResponse

Used to indicate if an XML response is to be returned.

Member

Value

Description

None

0

No XML response is desired.

Standard

1

The default XML response will be returned.
