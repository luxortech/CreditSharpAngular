Assemble Method

 

Namespace: LuxorTech.CreditSharp.Extract

    

 

The Assemble method exposed by the ExtractData object is used to obtain native bureau data using the CreditSharp framework.  Two methods are available which both can be tasked to produce either a Json or Xml structured output.  If the segment list parameter is left null or not used, then the extracted output will consist of every populated segment that is present within the bureau input.  Otherwise, the extracted segments will be confined to just those indicated in the List parameter.

 

Method Overloads

 

Name

Description

Assemble(BureauData input)

Returns a string in the desired layout (Json or Xml) consisting of all native bureau elements found within an instance of a Equifax, Experian or TransUnion data set.

Assemble(BureauData input, List<string> segments)

Returns a string in the desired layout (Json or Xml) consisting of any native bureau elements within an instance of a Equifax, Experian or TransUnion data set that match the provided list of segment names.

 

Syntax

public string Assemble (BureauData input)

public string Assemble (BureauData input, List<string> segments))

 

Parameters:

 

input

Type: BureauData object reference

Contains a populated instance of a BureauData object.

segments

Type: List<string>

Contains the names of segments to be extracted from the input object.

 

It is acceptable to pass in a null segments reference.  In such a case, all data present within the input object will be emitted.

 

Example: Extract all elements.

 

using System.Diagnostics;

 

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Datamart;

using LuxorTech.CreditSharp.Datamart.Equifax;

using LuxorTech.CreditSharp.Datamart.Experian;

using LuxorTech.CreditSharp.Datamart.TransUnion;

using LuxorTech.CreditSharp.Extract;

 

namespace HelpDemos

{

  public class DemoExtract

  {

      public string GetBureauXml(string creditDataBuffer, BureauIds creditBureau, string licenseKey)

      {

          BureauData bureauData = null;

          string output = string.Empty;

 

          switch(creditBureau)

          {

              case BureauIds.Equifax:

                  bureauData = new EquifaxData(new BureauOptions(licenseKey));

                  break;

              case BureauIds.Experian:

                  bureauData = new ExperianData(new BureauOptions(licenseKey));

                  break;

              case BureauIds.TransUnion:

                  bureauData = new TransUnionData(new BureauOptions(licenseKey));

                  break;

          }

 

          try

          {

              bureauData.Parse(creditDataBuffer);

 

              ExtractData extractData = new ExtractData(new ExtractOptions(ExtractLayout.Xml));

              output = extractData.Assemble(bureauData);

          }

 

          catch (CreditSharpException ex)

          {

              Debug.WriteLine(LuxorTech.CreditSharp.Core.Utilities.ErrorStack(ex));

          }

 

          return output;

      }

   }

}

 

Example: Extract selected elements.

 

using System.Diagnostics;

 

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Datamart;

using LuxorTech.CreditSharp.Datamart.Equifax;

using LuxorTech.CreditSharp.Datamart.Experian;

using LuxorTech.CreditSharp.Datamart.TransUnion;

using LuxorTech.CreditSharp.Extract;

 

namespace HelpDemos

{

  public class DemoExtract

   {

      public string GetExperianElements(string experianData, string licenseKey)

       {

          BureauData bureauData = new ExperianData(new BureauOptions(licenseKey));

          string output = string.Empty;

 

          List<string> segments = new List<string>();

           segments.Add("Inquiry");

           segments.Add("Tradeline");

           segments.Add("RiskModel");

 

          try

           {

               bureauData.Parse(experianData);

              ExtractData extractData = new ExtractData(new ExtractOptions(ExtractLayout.Xml));

               output = extractData.Assemble(bureauData, segments);

           }

 

          catch (CreditSharpException ex)

           {

              Debug.WriteLine(LuxorTech.CreditSharp.Core.Utilities.ErrorStack(ex));

           }

 

          return output;

       }

   }

}
