BureauRequest Class

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The BureauRequest type is an abstract base class which defines the generic properties and methods for requesting credit bureau data.  Each of the three bureau specific types for submitting a inquiry (EquifaxRequest, ExperianRequest, and TransUnionRequest) are derived from this class, which can be convenient when trying to consolidate code by upcasting the bureau specific object to the more generic base type.

 

An extensive example of this functionality can be found in the Streamlining Code topic.

 

The references below outline the members and features that can be accessed through this generic base class.  As there is no public constructor for this class, it can only be used as a target for upcasting and streamlining bureau processing, as shown in the linked example.

 

This base class includes properties for parameters that can each be found within all bureaus' input profiles.  However, this does not mean they share the same range of values. All of these properties employ values unique to each bureau, except for ContractAmount, IsJointRequest, PermissiblePurposeUser and ReferenceNumber.
