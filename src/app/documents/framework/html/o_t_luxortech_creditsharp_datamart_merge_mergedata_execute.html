Execute Method

 

Namespace: LuxorTech.CreditSharp.Datamart.Merge

    

 

The Execute method examines various elements of the provided common data objects looking for instances of duplicate items. There are five types of credit data references that are deduped: Addresses, Employments, Identifications, Public Records and Tradelines.  The remaining data references are merely copied from the common inputs into the the output merge object.

 

Duplicate items can be removed from one, two or three bureaus in a single operation with different options for identifying and resolving duplicates within each bureau (inter) as well as between bureaus (intra).

 

Method Overloads

 

Name

Description

Execute(CommonData input)

Eliminates duplicates from a single normalized bureau.

Execute(CommonData input, CommonData input2)

Eliminates duplicates within and between two normalized bureaus.

Execute(CommonData input, CommonData input2, CommonData input3)

Eliminates duplicates within and between three normalized bureaus.

 

Syntax

public void Execute (CommonData input)

public void Execute (CommonData input, CommonData input2)

public void Execute (CommonData input, CommonData input2, CommonData input3)

 

Parameters:

 

input1

Type: A CommonData object reference

Contains the a reference to a normalized set of credit bureau data.

 

input2

Type: A CommonData object reference

Contains a second reference to a normalized set of credit bureau data, which is to be merge with the first.  A null reference is acceptable.

 

input3

Type: A CommonData object reference

Contains a third reference to a normalized set of credit bureau data, which is to be merged along with the other two.  A null reference is acceptable.

 

Example 1

 

using LuxorTech.CreditSharp.Datamart.Common;

using LuxorTech.CreditSharp.Datamart.Merge;

 

namespace HelpDemos

{

  public class DemoMerge

  {

      public MergeData MergeBureauData(CommonData input, CommonData input2, CommonData input3)

      {

          MergeOptions options = new MergeOptions();

          options.AllowJoint = false; // Keep primary and secondary data separated

 

          MergeData merge = new MergeData(options);

          merge.Execute(input, input2, input3);

 

          return merge;

      }

   }

}
