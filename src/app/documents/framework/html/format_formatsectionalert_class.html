FormatSectionAlert Class

 

Namespace: LuxorTech.CreditSharp.Format

    

 

The FormatSectionAlert class is used to manipulate how alert information is presented within the various CreditSharp format outputs.  Customization options include changing the display sequence of each member as well as organizing them into separate groups. (Custom grouping is only available when generating one of the visual layouts, either Html or Text.)

 

The FormatSectionAlert class inherits four properties from its FormatSection parent class and implements its own sort method as highlighted in the example at the end of this chapter.  Identification sections have just one predefined sort option and no existing grouping configurations.

 

Existing sort criteria

 

The Name column denotes the argument that should be passed to the DefineSort() method when choosing an existing sort configuration.

 

Name

Order by

System

first by XtOrigin.BureauId (ascending), then by XtIdentMatch (ascending). then by ServiceId (ascending), last by MessageType (ascending)

 

[this is the default option]

 

Note: Any custom sorts of Alert data will only be applied to Json or Xml outputs.  The contents of Alert members occupies a fixed location and sequence within Text and Html outputs.

 

Existing group criteria

Name

Groups (Selector and headers)

Grouping is not supported within Alert sections.

 

Custom Sort

This snippet demonstrates setting up a custom sort for the Json layout.

 

using System;

 

using LuxorTech.CreditSharp.Analyze;

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Datamart.Common;

using LuxorTech.CreditSharp.Datamart.Merge;

using LuxorTech.CreditSharp.Format;

 

namespace HelpDemos

{

  public class DemoFormat

  {

      public string CreateText(MergeData input, AnalyzeData aggregates)

      {

          FormatOptions options = new FormatOptions(FormatLayout.Json);

          options.Json.Partitions = Partitioning.Report;

          options.Json.ReportStyle = JsonStyle.Common01;

 

          //

          // Obtain a reference for the configuration of an alert section in the Json layout

          //

          FormatSectionAlert section = (FormatSectionAlert) options.Json.Section("Alert");

 

          // Prepare the section for accepting a customized setting

          section.DefineSort("Custom");

          section.AddSort(x => x.ServiceId);

          section.AddSort(x => x.XtOrigin.BureauId);

 

          FormatData format = new FormatData(options);

          return format.Generate(input, aggregates);

      }

   }

}
