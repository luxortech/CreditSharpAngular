Custom Attributes

 

    

 

The CreditSharp Framework can be used to define and generate custom attributes that can then be used to drive proprietary score models or decision engines.

 

Attribute evaluation is automatically optimized to ensure the most efficient execution possible while offering the consistency and flexibility to meet just about any business need.  There are a number of built-in operations and features which can speed development while also eliminating much of the guesswork in handling sometimes intractable credit data conditions.

 

The four core credit data types available for custom analysis are: Identification, Inquiry, Public Record and Tradeline.  Along with being able to analyze these normalized credit data elements, the capability also exists to construct rules that reference the results from other attribute calculations.
