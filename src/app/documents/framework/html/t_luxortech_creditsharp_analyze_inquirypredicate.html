InquiryPredicate Delegate

 

Namespace: LuxorTech.CreditSharp.Analyze

    

 

Note: Custom evaluations are not yet fully supported.  This feature will be enabled in a soon to be released CreditSharp update.

 

The InquiryPredicate defines the method signature required to perform a true/false test in support of a compute operation.  It is used as part of a custom Analyze implementation. The delegate can either be a LINQ expression or a reference to a named method.  The logic provided is used to determine if a ComInquiry instance should be evaluated.
