Value

Description

AnalysisLimitProxy

Specifies if an alternate CreditLimit source can be used in relevant calculations.

 

Member

Value

Description

None

0

No proxy value should be used, calculation will not be performed without an explicit CreditLimit value.

Implied

1

AnalysisOperation

Indicates what type of instruction will be performed in a given evaluation.

 

Member

Value

Description

Count

0

Increments total by one if given segment matches search criteria.

Sum

1

Accumulates total value of targeted data element.

Average

2

Calculates the average value for all occurrences of the target data element.

Highest

3

Preserves the highest value found amongst the targeted data element.

Lowest

4

Preserves the lowest value found amongst the targeted data element.

Minimum

5

Also preserves the lowest value found amongst the targeted data element, but ignores the AssumeZeroValue option setting and always skips over blank values.

Maximum

6

Also preserves the highest value found amongst the targeted data element, but ignores the AssumeZeroValue option setting and always skips over blank values.

Newest

7

Preserves the most recent date found amongst the target data element.

Oldest

8

Preserves the oldest date found amongst the target data element.

Compute

9

Solves an equation using references to one more previously calculated values.

 

Review the Operations topic pages for examples and discussion of each type of operation.

Specifies how the Open or Closed status of a given Tradeline is decided.

 

Member

Value

Description

Strict

0

Bases the open/closed decision solely upon the value of the ClosedIndicator

Inferred

1

Lenient
