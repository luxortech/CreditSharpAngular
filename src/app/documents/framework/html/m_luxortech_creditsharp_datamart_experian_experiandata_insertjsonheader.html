InsertJsonHeader Method

 

Namespace: LuxorTech.CreditSharp.Datamart.Experian

    

 

The InsertJsonHeader method injects header information (date credit was obtained and, if desired, a user defined reference number) into an existing JSON formatted Experian credit bureau response.

 

Experian does not currently embed the date in which the credit data ordered within their JSON responses. Since that value is vital when performing date-based arithmetic during credit analysis, this method can be used to supply that detail.

 

Note, this is only required if the credit data was obtained outside of the CreditSharp framework. If Experian JSON credit responses are requested using CreditSharp, then this requisite header information is inserted into the JSON bureau response at the time the inquiry is made before returning it to the caller.

 

 

Syntax

public string InsertJsonHeader (DateTime creditReportDate, string referenceNbr, string creditDataBuffer)

 

Parameters:

 

creditReportDate

Type: DateTime

Indicates the date the credit report was originally obtained

referenceNbr

Type: string [optional]

Contains a unique value used by an organization to identity a report (such as a prospective loan number).  If this value is not going to be used it can simply be passed as an empty string.

 

creditDataBuffer

Type: string

The JSON formatted credit bureau data to be augmented.

 

Exceptions

Exception class

Status

BureauDataException

BureauDataStatus.CreditDataInvalid

 

Example

 

using LuxorTech.CreditSharp.Datamart;

using LuxorTech.CreditSharp.Datamart.Experian;

 

namespace HelpDemos

{

  public class DemoFormat

  {

      //

      // Generate HTML report using Experian JSON input which does not already include header info.

      //

      // This example does not include a reference number so that parameter is passed as empty.

      //

      public string GetExperianReport(string licenseKey, string creditData, DateTime requestDate)

       {

          ExperianData experianData = new ExperianData(new BureauOptions(licenseKey));

          experianData.Parse(experianData.InsertJsonHeader(requestDate, "", creditData));

 

          CommonData commonData = new CommonData(new CommonOptions());

           commonData.Normalize(experianData);

 

          FormatData formatData = new FormatData(new FormatOptions(FormatLayout.Html));

          return formatData.Generate(commonData);

       }

    }

}
