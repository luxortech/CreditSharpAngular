Public Record Status Codes

 

    

 

Public Record Status Codes indicate the current disposition of a particular legal filing.  These values can be used to determine if a legal action is still pending or has been completed.  A pending status might indicate that the consumer still has a potential liability depending on the outcome of the case.  

 

When analyzing Public Record data for additional consumer debt, only items with a logical "pending" status should be included in those calculations.  The "Disposition" column in the following indicates whether the Status Code equates to a logical "pending" state or a logical "concluded" state.  When "concluded", a legal item no longer carries any potential financial burden for the defendant (presumably, the consumer on whose credit report it appears.)
