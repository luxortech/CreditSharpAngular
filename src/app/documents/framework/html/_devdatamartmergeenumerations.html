CompareMethod

Indicates how duplicate items are identified.

 

Member

Value

Description

None

0

No attempt is made to merge the referenced items.

Basic

1

The most restrictive duplicate detection criteria - essentially looks only for exact matches.

Standard

2

A more lenient level of criteria wherein more interpretive comparisons allowing a wider variation between data elements - most appropriate for locating duplicate items within a single bureau.

Advanced

3

The most expansive collection of evaluations which detects a larger range of duplicate items - best used for eliminating duplicate between different credit bureaus.

 

CreditSharp defaults to using Standard for intra-bureau merging and Advanced for inter-bureau merging. Note that, at this time, the Advanced level is only effectively supported with Tradeline data.  However, over time, future enhancements will add more data elements to those with advanced treatments.
