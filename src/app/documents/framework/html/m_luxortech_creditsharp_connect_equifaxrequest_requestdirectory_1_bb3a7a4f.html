EquifaxRequest.RequestDirectory Method

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The RequestDirectory method is used to obtain more detailed name, address and phone number information for a specific Equifax data contributor.  The returned value is an instance to the BureauSubscriber class which contains whatever detailed information Equifax was able to locate for the input subscriber.

 

This feature has not yet been implemented, but will soon be available in an upcoming CreditSharp release.

 

Syntax

public BureauSubscriber RequestDecode(string memberNumber)

 

Parameters:

 

memberNumber

Type: string

Contains the bureau specific id for the data provider being decoded.

 

Example

 

      public void GetSubscriberInfo(string memberNumber)

      {

          BureauConnection connection = new BureauConnection(BureauIds.Equifax);

          connection.HttpSettings.ServiceName = "acrotest";

          connection.HttpSettings.ServiceURL = "https://transport5.ec.equifax.com/servlet/stspost";

          connection.HttpSettings.WebUserId = "myCompanyId";

          connection.HttpSettings.WebPassword = "letMeIn";

 

          EquifaxRequest request = new EquifaxRequest(new EfxInquiryOptions(licenseKey));

          request.ResponseVersion = EfxVersionNumber.Version50;

          request.SubscriberCode = "222XX11111";

          request.SubscriberPassword = "ABC";

 

          BureauSubscriber bureauSub = request.RequestDirectory(memberNumber);

 

          return;

      }
