TucLookup Class

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The TucLookup class is used to provide the required inputs for requesting detailed contact information for a given bureau subscriber.

 

Properties

 

Name

Content

Description

SubscriberIndustry

Char (2)

The Industry code for the current subscriber.

SubscriberMarket

Text (2)

The Market code to which the given subscriber belongs.

 

This value is only required when the SubscriberType is IN.

SubscriberNumber

Text (8)

The Subscriber number to be decoded.

SubscriberSubmarket

Text (2)

The Sub-Market code to which the given subscriber belongs.

 

This value is only required when the SubscriberType is IN.

SubscriberType

Char (2)

The Subscriber Type (if any) to be decoded.  This entry indicates from what segment the given subscriber code was reported.

 

Values:

CL

A collection subscriber (from a CL01 segment)

IN

An inquiry subscriber (from an IN01, IN02 segment)

PR

A public record subscriber (from a PR01 segment)

TR

A tradeline subscriber (from a TR01 segment)
