EfxInquiryOptions Class

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The properties presented within the EfxInquiryOptions class are specific to an Equifax credit bureau request.  They are used to request various forms of supplemental information be added to a credit report response.

PdfWaitTime

Num (2)

Gets or sets the value indicating how many attempts will be made to automatically retrieve a PDF credit report image from the new Equifax cloud based API.

 

In instances where the Equifax server(s) are abnormally busy they may be a appreciable delay when creating a printable PDF image of a consumer credit report to accompany the native JSON raw data response.  This PdfWaitTime property is used by the CreditSharp framework to repeatedly attempt to obtain the PDF image, for the number of times indicated here, with a one second interval between each attempt.

 

If the PDF cannot successfully be obtained in the allotted time, the EquifaxRequest.PdfStatus property will be assigned one of several error code values indicating why the attempt(s) failed.

 

[Defaults to 4] (ie: up to four attempts will be made to retrieve the PDF pausing one second between each try.
