Properties

 

Name

Content

Description

IdDescription

Text (20)

Gets or sets the value providing a description of the identification element.

 

Optional input element which can typically be excluded, except when providing Credit Card information.  In that instance, the name of the company which issued the credit card should be supplied in this property..

IdExpirationDate

Text (10)

Gets or sets the value specifying when the id will no longer be valid.

 

Format:

Entered in month, day, year order. At minimum, the input must include the month and year.  It can contain standard date delimiters.  For example, March 16th 1997 can be input as:

03-1997, 031997, 03/1997, 03/16/1997, 03-16-1997, 03161997.

IdIssueDate

Text (10)

Gets or sets the value specifying when the id was issued.

 

Format:

Entered in month, day, year order. At minimum, the input must include the month and year.  It can contain standard date delimiters.  For example, March 16th 1997 can be input as:

03-1997, 031997, 03/1997, 03/16/1997, 03-16-1997, 03161997.

IdIssueState

Char (4)

Gets or sets the value specifying in which state the id was issued.

IdNumber

Text (20)

Gets or sets the value number corresponding the the given id.

 

This might be the Driver's License number, the credit card number, etc.

IdType

Enum

Indicates the type of supplemental information being supplied.
