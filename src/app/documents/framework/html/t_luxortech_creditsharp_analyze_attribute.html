AnalyzeAttribute Class

 

Namespace: LuxorTech.CreditSharp.Analyze

    

An instance of the AnalyzeAttribute type provides the outcome of a given evaluation.  These evaluation outcomes are found within the collection exposed by the Results property of the AnalyzeData class.

 

Properties

 

Name

Content

Description

 

HasValue

Bool

Gets the flag indicating whether the current attribute was calculated.

 

When true, the attribute contained an actual computed value.  When false, no qualifying elements where found that could contribute to the result.

 

A false indication here, for example, could be for an evaluation looking to obtain the highest mortgage payment amount.  If there are no mortgages in the active bureau data, then the computed value will be zero, but, the HasValue property will be false to indicate that the result is an implicit one.

 

Note that the text based Value property will be an empty string if HasValue is false and the AnalyzeOptions.DefaultEmptyCalcs property was changed to false.

 

Id

Num

Gets the unique numeric identifier for the current attribute.

 

IsDate

Bool

Indicates if the attribute value contains an integer representation of a date.

 

When true, the content is a date value resulting from a Newest or Oldest operation.

 

The corresponding quantity and value holds the numeric equivalent of a "YYYYMMDD" date string.  For example, the date June 11th, 1963 would be rendered as the number 19630611 (or 19,630,611).

 

The helper function CreditSharp.Core.Utility.IntToDate() can be used to convert this into a .NET DateTime value.

 

Name

Text

Gets the string which logically identifies the attribute.

 

Observations

Text

Get a string of semi-colon delimited numeric values indicating which segments were included in the calculations for the current attribute.

 

The numeric value corresponds to the DtSequence property of the corresponding datamart instance(s) that were evaluated.

 

Operation

Gets the AnalysisOperation enumeration indicating what type of calculation was used to generate the current attribute.
