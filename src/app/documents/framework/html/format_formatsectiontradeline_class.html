FormatSectionTradeline Class

 

Namespace: LuxorTech.CreditSharp.Format

    

 

The FormatSectionTradeline class is used to manipulate how consumer tradeline information is presented within the various CreditSharp format outputs.  Customization options include changing the display sequence of each member as well as organizing them into separate groups. (Custom grouping is only available when generating one of the visual layouts, either Html or Text.)

 

The FormatSectionTradeline class inherits four properties from its FormatSection parent class and implements its own sort and group methods as highlighted in the examples at the end of this chapter.  Tradeline sections have two predefined sort options available along with a several grouping configurations.

Example 1: Enable a predefined group

This snippet demonstrates enabling the OpenClosed group for use in an Html layout.

 

      private void CustomizeTradeGroups(FormatOptions options)

      {

          FormatSectionTradeline section = (FormatSectionTradeline)options.Text.Section("Tradeline");

 

          // Select the group configuration to be used when generating this Html output

          section.DefineGroups("OpenClosed");

      }

 

Example 2: Modify Headers

This snippet demonstrates selecting the AcctType group in order to disable the Missing headers and shifting the Initial headers to mixed case.

 

      private void CustomizeTradeGroups(FormatOptions options)

      {

          TextInfo textInfo = new CultureInfo("en-US",false).TextInfo;

 

          FormatSectionTradeline section = (FormatSectionTradeline)options.Text.Section("Tradeline");

 

          // Select the group to be adjusted

          section.DefineGroups("AcctType");

 

          FormatGroupTradeline group = section.Group("Col");

          group.MissingHeader = string.Empty;

          group.InitialHeader = textInfo.ToTitleCase(group.InitialHeader);

 

          group = section.Group("Mtg");

          group.MissingHeader = string.Empty;

          group.InitialHeader = textInfo.ToTitleCase(group.InitialHeader);

 

          group = section.Group("Ins");

          group.MissingHeader = string.Empty;

          group.InitialHeader = textInfo.ToTitleCase(group.InitialHeader);

 

          group = section.Group("Rev");

          group.MissingHeader = string.Empty;

          group.InitialHeader = textInfo.ToTitleCase(group.InitialHeader);

 

          group = section.Group("Unk");

          group.MissingHeader = string.Empty;

          group.InitialHeader = textInfo.ToTitleCase(group.InitialHeader);

      }

 

Example 3: Custom Sort and Group

This snippet splits a Html tradeline section into two groups: Auto loans and Not auto loans.

 

      private void CreateCustomGroup(FormatOptions options)

      {

          FormatSectionTradeline section = (FormatSectionTradeline)options.Html.Section("Tradeline");

 

          // Define custom sort: order by descending balance amount

          section.DefineSort("Custom");

          section.AddSort(x => x.AMT(ComTradelineIds.BalanceAmount) * -1);

 

          // Create two groups:  Auto loans and Not-Auto loans

           section.DefineGroups("Custom");

 

          FormatGroupTradeline group = section.AddGroup("Auto");

          group.InitialHeader = "AUTO LOANS";

          group.MissingHeader = "AUTO LOANS: <span style=\"color:red\">None found</span>";

 

          group.Selector = (x => x.IsAuto);

 

          group = section.AddGroup("Other");

          group.InitialHeader = "OTHER ACCOUNTS";

          group.MissingHeader = "OTHER ACCOUNTS: <span style=\"color:red\">None found</span>";

 

          group.Selector = (x => !x.IsAuto);

      }
