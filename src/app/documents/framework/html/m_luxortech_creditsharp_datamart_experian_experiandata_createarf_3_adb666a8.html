ExperianData.CreateArf Method

 

Namespace: LuxorTech.CreditSharp.Datamart.Experian

    

 

The CreateArf method will generate a Automated Response Format layout using whatever bureau image is present within the current ExperianData object.  The can be useful when an existing system can only process the ARF Experian format but needs to accept data from providers that is in different Experian formats.

 

Using this, for example, an Experian XML bureau response can be imported into the CreditSharp Framework and then exported into an equivalent Automated Response Format image.

 

Method Overloads

 

Name

Description

CreateArf(ExpVersionNumber targetVersion)

Converts the currently imported bureau content into a Automated Response Format image of the requested version.

CreateArf(ExpVersionNumber targetVersion, bool maskIdentity, string segmentDelimiter)

Converts the currently imported bureau content into a Automated Response Format image of the requested version. Additionally, sensitive information can be obscured and individual data segments can be divided using the provided delimiter (such as a NewLine).

 

Syntax

public string CreateArf (ExpVersionNumber targetVersion)

public string CreateArf (ExpVersionNumber targetVersion, bool maskIdentity, string segmentDelimiter)

 

Parameters:

 

targetVersion

Type: ExpVersionNumber enumeration

Specifies which version of the Full File Fixed layout should be created.

maskIdentity

Type: bool

Instructs operation to scramble consumer name and identification number (SSN, SIN) as well as tradeline account numbers.

Note: this feature has not yet been implemented

 

segmentDelimiter

Type: string

When provided, the delimiter will be appended to each segment that is created.  This results in an output that cannot be processed or recognized by other systems, but, can be useful when performing research or troubleshooting.  

 

The best example of this would be to provide a New Line character as a delimiter, so that the resulting output can then be reviewed in an editor.

 

Example

 

using LuxorTech.CreditSharp.Core;

using LuxorTech.CreditSharp.Datamart;

using LuxorTech.CreditSharp.Datamart.Equifax;

using LuxorTech.CreditSharp.Datamart.Experian;

using LuxorTech.CreditSharp.Datamart.TransUnion;

 

namespace HelpDemos

{

  public class DemoCreate

  {

      public string ConvertXmlImage(string bureauXml, BureauIds bureauId)

      {

          string cpuOutput = string.Empty;

 

          switch (bureauId)

          {

              case BureauIds.Equifax:

                  EquifaxData equifaxData = new EquifaxData(new BureauOptions(licenseKey));

                  cpuOutput = equifaxData.CreateFull(EfxVersionNumber.Version60);

                  break;

              case BureauIds.Experian:

                  ExperianData experianData = new ExperianData(new BureauOptions(licenseKey));

                  cpuOutput = experianData.CreateArf(ExpVersionNumber.Version70);

                  break;

              case BureauIds.TransUnion:

                  TransUnionData transUnionData = new TransUnionData(new BureauOptions(licenseKey));

                  cpuOutput = transUnionData.CreateFfr(TucVersionNumber.Version40);

                  break;

          }

          return cpuOutput;

      }

   }

}
