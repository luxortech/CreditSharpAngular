AddModel Method

 

    

 

The AddModel method is used to inject a custom score model definition for use when parsing and subsequently normalizing the current credit bureau data.  The method returns an instance of the core BureauScore class which can then be used to append any error code or adverse action codes with their corresponding text translations.

 

For more detail usage information please refer to the BureauScore topic page.

 

Note: These definitions can also be added to the CreditSharp.dll.config file to automatically load the custom models and avoid having to perform any additional coding tasks
