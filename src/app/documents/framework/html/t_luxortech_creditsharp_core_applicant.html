Applicant Class

 

Namespace: LuxorTech.CreditSharp.Core

    

 

Root class for supplying the details about an applicant to be queried at a credit bureau.

 

Constructor

 

Name

Description

Applicant()

Class constructor for the root object for specifying a consumer to be investigated.

 

Syntax

public Applicant()

 

Parameters: none
