BureauDataStatus Enumeration

 

    

 

Defines group of possible CreditSharp generated exceptions that may be encountered using the BureauData or CommonData classes.

 

Members

Name

Value

Description

SystemException

0

Indicates a system exception has been caught and rethrown. Inspect the Inner exception for detail.

InputBufferEmpty

1

Thrown when creditDataBuffer input parameter is empty.

InfiniteLoop

2

Thrown when normal processing would result in an infinite loop.

CreditDataUnidentified

3

Thrown when creditDataBuffer input parameter does not contain an identifiable credit response.

CreditDataInvalid

4

Thrown when creditDataBuffer input parameter contains an unidentifiable element.

CreditDataTruncated

5

Thrown when creditDataBuffer input terminates prematurely.

UnsupportedBureauVersion

6

Thrown when creditDataBuffer contains a response version that is not currently supported.

CreditDataUnterminated

7

Thrown when creditDataBuffer input parameter is not properly terminated.

MissingXMLAttribute

8

Thrown when a required attribute values is missing from an XML creditDataBuffer input.

MissingXMLEndTag

9

Thrown when an XML element is not properly terminated.

UndefinedXMLEnumeration

10

Thrown when an XML value is not a recognized enumeration.

InvalidDecisionSchema

11

Thrown when a JSON formatted Decision Schema could not be parsed or contains an invalid entry.

FeatureNotSupported

12

Thrown when invoking a method or referencing a property which is not currently implemented.

PdfImageDoesNotExist

13

Thrown when requesting a Pdf image (via the Image() method) but none exists.
