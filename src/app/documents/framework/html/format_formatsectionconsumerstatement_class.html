FormatSectionConsumerStatement Class

 

Namespace: LuxorTech.CreditSharp.Format

    

 

The FormatSectionConsumerStatement class is used to manipulate how consumer statement information is presented within the various CreditSharp format outputs.  Customization options include changing the display sequence of each member as well as organizing them into separate groups. (Custom grouping is only available when generating one of the visual layouts, either Html or Text.)

 

The FormatSectionConsumerStatement class inherits four properties from its FormatSection parent class and implements its own sort and group methods as highlighted in the examples at the end of this chapter.  Consumer statement sections have one predefined sort option and no existing grouping configurations.

Custom Group

This snippet splits a Html consumer statement section into two groups: Alert statements and other statements.

 

      private void CreateCustomGroup(FormatOptions options)

      {

          FormatSectionConsumerStatement section = (FormatSectionConsumerStatement)options.Text.Section("ConsumerStatement");

 

          // Create two groups:  Alert statements and all others

          section.DefineGroups("Custom");

 

          FormatGroupConsumerStatement group = section.AddGroup("Alert");

          group.InitialHeader = "ALERT NOTICE";

 

          group.Selector = (x => x.CategoryCode.In(4, 5, 6));

 

          group = section.AddGroup("Other");

          group.InitialHeader = "GENERAL STATEMENT";

 

          group.Selector = (x => !x.CategoryCode.In(4, 5, 6));

      }
