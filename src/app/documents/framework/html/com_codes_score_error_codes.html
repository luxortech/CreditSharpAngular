Score Error Codes

 

    

 

The following tables list the possible error codes that may be returned when ordering a credit report scoring model.  The codes are unique to each credit bureau.  The error descriptions provided here are also displayed in the ComScore segment's ErrorText element.

 

For Equifax, codes are provided for the Fico/Beacon and Das score models.  Error codes for all other Equifax scores are vary depending upon the model.  In those instances, refer to the ComScore segment's ErrorText element for an error description.
