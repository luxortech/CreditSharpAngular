ExpInquiryOptions Class

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The properties presented within the ExpInquiryOptions class are specific to an Experian credit bureau request.  They are used to request various forms of supplemental information be added to a credit report response.

Description

Parameters:

licenseKey

Type: string

The 36 character key which grants access to all CreditSharp features.
