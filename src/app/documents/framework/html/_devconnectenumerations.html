Connect Enumerations

 

Namespace: LuxorTech.CreditSharp.Connect

    

 

The following tables list all the enumerated values available for use when setting various properties found within the Connect namespace.

 

Enumeration

Description

ConnectionType

Indicates what communication protocol will be used.

 

Member

Value

Description

Http

0

Communications will be carried out using Http connection protocols. (Default)

Tcpip

1

Communications will be performed over a dedicated Tcp/ip channel.

Sna

2

Communications will be conducted using System Network Architecture constructs.

 

Tcpip and Sna are not currently supported.

 

ProxyType

Indicates if Http communications will be performed through a proxy server.

 

Member

Value

Description

None

0

No internet proxy will be used. (Default)

Configured

1

Proxy information will be obtained from the default system configuration.

Named

2

Proxy information will be applied from provided ProxyServer inputs.

ScoreDisclosure

Used to specify if an Experian score disclosure form should be embedded within the credit bureau response.

 

Member

Value

Description

None

0

Age No disclosure form generated. (Default)

Mortgage

1

Include the Experian Mortgage disclosure form, to be used when credit application is for a mortgage loan.

Generic

2

Include the Experian Generic disclosure form, to be used when credit application is for anything but a mortgage loan.

 

As implied above, this option is only intended for Experian score requests.  The setting can only be applied to a single bureau score, so it should be assigned to the score being used in the primary lending decision process.

ScoreSupplement

Indicates if supplemental consumer score disclosure elements are to be included in the bureau response.

 

Member

Value

Description

None

0

No supplemental score details will be returned. (Default)

Range

1

Indicates that minimum and maximum scores for a given model are to be returned.

Rank

2

Indicates that the score's relative position (percentile) within all scores for a given model is to be returned.

RangeAndRank

3

Indicates that both Range and Rank should be returned.

 

This model-by-model setting is only applicable with TransUnion score model requests.  Equifax and Experian both have equivalent settings, but are applied globally to all score models using the SupplementalScoreData property found in EfxInquiryOptions or ExpInquiryOptions.
