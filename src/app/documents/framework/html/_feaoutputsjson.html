JSON Output

 

    

 

Outputs from the CreditSharp framework can be obtained in the popular Java Script Object Notation (JSON) syntax.

 

The contents of JSON formatted outputs can be manipulated via settings within the FormatOptionsJson object.  It is here that certain elements can be included (placeholders for null fields, for example) or excluded (entire classes or data, such as Directory information, can be hidden from view).  

 

The JSON output is modeled after the normalized object(s) from within the CommonData namespace.  Instances of each common data class will appear within the JSON stream as an array of objects corresponding to each given type.  The table below outlines this relationship.
