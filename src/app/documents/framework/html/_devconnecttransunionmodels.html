TransUnion Score Models

 

    

 

 

The table below lists all TransUnion models currently recognized by the CreditSharp framework.

 

If a score model needs to be ordered that does not appear in this list, it can be preceded with an exclamation point to suppress the internal validation which prevents unknown models from being requested.  For example, TransUnionRequest.AddModel("!08872") would successfully append the hypothetical new model 08872 to the inquiry.
