Using Properties to request custom solutions

 

    

 

The following custom Experian products can be requested from Experian using existing property settings found in the ExpInquiryOptions class.

 

 

Extended View

 

To obtain one of the Extended View score or attribute outputs, simply enable the desired response by assigning the appropriate ExpExtendedView enumeration to the ExtendedViewResponse property.

Prequalification

 

Assign the ExpProductCode.Preapproval enumeration to the ExperianRequest.ProductCode property.

 

When ordering the Standard prequalification product all that is required is the appropriate ExperianRequest.PermissiblePurposeCode (typically "3F").  

 

When ordering any other prequalification product supply the Experian provided values for the ExpInquiryOption.CustomRRKeyword, such as PSOQ, and ExpInquiryOption.PrequalificationSolutionId properties.

 

 

WESCOM custom solution

 

Assign the ExpProductCode.Service enumeration to the ExperianRequest.ProductCode property.

 

Assign the value "WSCOM" to the ExpInquiryOption.CustomRRKeyword for submitting production/live inquiries. If sending a test inquiry, use "XWSCOM" instead.

 

Assign any associated scoring models, such as "A1", using the ExperianRequest.AddModel method.
