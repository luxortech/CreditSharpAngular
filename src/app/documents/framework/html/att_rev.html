REV (Revolving) Attributes

 

    

 

The Revolving attribute category includes values calculated with references to variable payment tradeline account types, typically restricted to credit cards.  As such, Credit Line and Open/30 Day account information is not reflected in this category. (See the CRL or OPN category).  If information for all three types needs to be considered together, then use VAR category.
