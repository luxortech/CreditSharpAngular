Risk Based Pricing Updates

 

    

 

For any given bureau score model, the minimum and maximum possible scores and a consumer's rank compared to other individuals (the score percentile) is not always returned in a credit bureau response.  In order to fill in any missing information an internal set of tables are maintained which can be used to look up the any missing details for a given score.

 

These internal tables are updated on an annual basis, as the new score distributions are published by each bureau.  The chart below lists when the annual updates have been applied and cross references the legacy Toolkit software platform for co-ordination purposes.
