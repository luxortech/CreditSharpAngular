TEXT Sample

 

    

 

The are two basic TEXT formats, Common01 and Lending01.  The Common01 layout organizes the output into two logical reports, one for each applicant.  The Lending01 layout has the same appearance but groups all primary and secondary applicant items together.

 

The sample shown below represents the output for the Common01 layout.  The Lending01 layout is more suited to displaying merged data sets.
