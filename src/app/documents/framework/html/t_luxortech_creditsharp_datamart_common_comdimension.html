ComDimension Class

 

Namespace: LuxorTech.CreditSharp.Datamart.Common

    

 

The ComDimension class presents an instance of historical tradeline activity highlighting payments and conditions for a specific month in a given account's past.

 

There may be one or more instances of this class in a List<ComDimension> which is exposed via the ComTradeline Dimensions property.  In cases where there are multiple dimension occurrences, the first one in the list will be the most recent, and each subsequent item will contain progressively older items.  The record of historical activity will not necessarily be continuous, there may be missing months so the ObservationDate should be noted when reviewing individual items.

 

Note:

These dimension elements are only populated when the credit data conatins a bureau's enhanced data response.  For Equifax, this is known as 'Total View Dimensions'.  For Experian, the service is known as Trend View.  For TransUnion, the output is included in the Account History portion of the CreditVision extensions.
